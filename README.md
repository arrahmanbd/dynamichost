## Project Task: Login & Dashboard with Animation

### Overview
This project implements a simple **Login Page** and **Dashboard Page** with a **menu grid that expands with a smooth animation**, inspired by the Bkash app.  

The UI is designed for demonstration purposes and shows how expandable menus and animated components can enhance user experience.

---

### Features

1. **Login Page**
   - Standard login with email/username and password.
   - Clean and professional UI.
   - Easy to integrate with backend authentication.

2. **Dashboard Page**
   - Grid-based menu layout.
   - **Expandable menu section** inspired by Bkash’s "See More" button.
   - Smooth height animation when toggling additional menu items.
   - Responsive layout with rounded cards and clean spacing.
   - Expanding menu provides visual feedback to users.
   - Active/inactive indicators and buttons styled professionally.

3. **Slider**
   - Uses `Slider` for smooth sliding transitions.

### Notes
- The expandable menu in the dashboard **does not exactly replicate Bkash** UI but mimics the functionality using an animated toggle.
- Designed to be **reusable and easy to customize**.
- Built using **Flutter**, can be easily integrated with Riverpod, Hooks, or any state management solution.
